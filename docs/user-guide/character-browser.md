# Community Character Browser

Browse, download, and share complete character builds with the community.

## Overview

Share your character builds and download builds from other players through the integrated community browser.

Located inside of the Character Management tab

**Features:**

- Browse characters by level, build type, popularity
- Download complete characters with one click
- Upload your characters with screenshots
- Like and track downloads
- Report inappropriate content
- Local caching for offline access
- Convergence mod support

## Downloading Characters

### Steps

1. Browse or search for character
2. Click character card to view details
3. Review full stats and equipment
4. Select Character slot of loaded save file
4. Click **Download & Import**

**What Gets Downloaded:**

- Complete character data
- All inventory
- Quest progress
- Appearance

**Automatic Adjustments:**

- SteamID patched to your account
- Checksums recalculated
- Backup created before import

### Character Details View

**Displays:**

- Full stats (Vigor, Mind, Endurance, etc.)
- Equipment loadout
- Build description
- Uploader notes
- Preview images (face, body, full)
- Download/like counts

### Download Stats

**Tracked:**

- Total downloads
- Likes

## Uploading Characters

### Requirements

- Character in valid slot
- Appropriate content (no offensive names/appearance)
- Must have a Github account and be logged into GitHub

### Steps

1. Load save with character to share
2. Go to the **Character Management** tab and click the **Browse Character Library Button**
3. Click **Contribute Character**
4. Select character slot
5. Fill in upload form:
   - Display name
   - Build description
   - Build category
   - Tags
6. Add images
7. Review preview
8. Click **Submit**
9. A Browser Window opens, drag and drop the prepared zip into it and Submit
10. Wait for maintainer to review and merge

**Result:** Character uploaded, visible to community

### Upload Form Fields

**Required:**

- Character name
- Author Name
- Build description
- Tags
- Screenshots

**Convergence/other overhaul mods:**

- Checkbox if modded character
- Select mod and version

### How to Report

1. View character details
2. Click **Report** button
3. Select reason
4. Add details

## Best Practices

### For Uploaders

**Good descriptions:**

- Explain build concept
- List key equipment
- Note playstyle
- Mention quest requirements

**Good tags:**

- Accurate build type
- PvP/PvE designation

**Good images:**

- Clear face view
- Full body visible

### For Downloaders

**Before downloading:**

- Read description fully
- Check equipment requirements
- Verify level range
- Read uploader notes

**After downloading:**

- Test character in-game
- Leave feedback (likes)
- Report if issues
- Share if enjoyed

## Troubleshooting

### "Character failed to download"

**Solutions:**

- Check internet connection
- Verify server status
- Try again later
- Clear cache and retry

### "Downloaded character won't load"

**Solutions:**

- Check SteamID patched correctly
- Run Save File Fixer
- Verify checksums
- Restore from backup

### "Upload failed"

**Solutions:**

- Check file size (<30MB)
- Verify description meets minimum
- Ensure images generated
- Check internet connection

### "Can't find character I uploaded"

**Solutions:**

- Refresh browser
- Check pending approval status
- Verify upload completed
- Check moderation queue

## FAQ

**Q: Are downloads safe?**  
A: Characters are scanned for corruption. SteamID auto-patched.

**Q: Can I upload multiple characters?**  
A: Yes, no limit on uploads.

**Q: Do I need an account?**  
A: No account needed for downloading. Github account needed for uploading.

**Q: Can I update uploaded characters?**  
A: No, you must upload new version with updated name.
